## Installation (Android)

1.  Download APK: https://lflauncher.vercel.app/
2.  Enable **Install unknown apps**
3.  Install **LFLauncher.apk** and open the app

**Notes:** - Requires \~**200MB** free storage - Login / activate
LFLauncher account - Reinstall if installation fails

## Minecraft Versions

-   Go to **Settings / Versions**
-   Select version ‚Üí **PLAY**
-   Add new version with **+ Add Version**

**Notes:** - \~**100--150MB** per version - Worlds & settings stay safe

------------------------------------------------------------------------

## Redeem Codes

STORE ‚Üí **Redeem Code** ‚Üí Enter code ‚Üí **REDEEM**

**Notes:** - One use per account - Codes may expire

## Troubleshooting

**App crashes:** - Close background apps - Restart device - Clear cache
/ reinstall

**Game won't start:** - Re-download version - Grant storage permission -
Free up space
