## Installation (Android)

1. Download APK: https://lflauncher.vercel.app/  
2. Enable **Install unknown apps**  
3. Install **LFLauncher.apk** and open the app  

**Notes:**  
- Requires ~**200MB** free storage  
- Login / activate LFLauncher account  
- Reinstall if installation fails
