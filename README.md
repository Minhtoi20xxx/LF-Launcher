# 1. How to Install LFLauncher on Android

### Step 1: Download the APK
Visit the official website to download the latest version:  
**https://lflauncher.vercel.app/**

### Step 2: Enable Unknown Sources
Go to:

**Settings ‚Üí Security ‚Üí Enable ‚ÄúUnknown sources‚Äù**  
or  
**Settings ‚Üí Apps ‚Üí Install unknown apps**

Some devices require enabling install permission for your **browser** or **file manager**.

### Step 3: Install the APK
1. Open your **Downloads** folder.  
2. Tap **LFLauncher.apk** and follow the instructions.  
3. After installation, open LFLauncher to get started.

**Notes:**  
- Ensure at least **200MB** of free storage.  
- You must log in or activate your **LFLauncher account**.  
- If installation fails, uninstall the old version and try again.

# 3. How to Use LFcoins and Shop in the Store

### What Are LFcoins?
LFcoins are the launcher‚Äôs **virtual currency** used to buy:  
- Maps  
- Skins  
- Texture packs  
- Other content packs

### How to Earn LFcoins
1. **Daily Login:** Earn free LFcoins each day.  
2. **Complete Missions:** Participate in events and missions.  
3. **Purchase Coins:** Buy extra coins directly in the Store.  
4. **Redeem Codes:** Apply gift codes to receive LFcoins.

### How to Buy Content
1. Go to **SHOP ‚Üí Store**.  
2. Browse or search for content packs.  
3. Select an item to view details.  
4. Tap **BUY NOW** and confirm using LFcoins.  
5. Your content is delivered instantly.

**Notes:**  
- Your LFcoin balance appears in the top-right corner.  
- Items are **non-refundable**.  
- LFcoins **cannot be transferred**.

# 6. Troubleshooting ‚Äî Game Fails to Launch

### Issue 1: ‚ÄúThe app keeps stopping‚Äù
**Cause:** Insufficient RAM or conflicting apps.  
**Solutions:**  
- Close background apps  
- Restart device  
- Clear LFLauncher **cache**  
- Reinstall the app

### Issue 2: ‚ÄúGame won't start‚Äù
**Cause:** Corrupted files or incompatible version.  
**Solutions:**  
- Remove the broken version  
- Re-download via Version Manager  
- Enable storage permissions  
- Ensure enough free storage
